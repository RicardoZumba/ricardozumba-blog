Ricardo Z

Los programadores de software libre no ofrecen una garantía de calidad suficiente. Soy Ricardo Fabián Zumba Ruiz, estudiante de la Universidad de Guayaquil, Facultad Ciencias matemáticas y físicas.

Introducción

El software son programas de cómputo y su documentación asociada: requerimientos, modelos de diseño y manuales de usuario. El software puede ser Genérico o Ad hoc.»

Ricardo Z 18 Agosto 2020

La Crisis del Software

Muchos de los proyectos de software presentan deficiencias como: Retraso en la entrega, Falta de fiabilidad, Coste excesivo, Ineficiencia, Mantenimiento problemático, Falta de adaptabilidad, Escasa portabilidad, Carencia de documentación, etc.»

Ricardo Z 18 Agosto 2020

Ingeniería del Software

La IS supone la aplicación del conocimiento científico al diseño y construcción de software y la documentación asociada requerida para su desarrollo, operación y mantenimiento.»

Ricardo Z 18 Agosto 2020

Objetivos

Planificar el desarrollo del software como un proceso metodológico de construcción y de gestión de proyectos soportado en estándares utilizados en la isdustria de la ingeniería de software para diseñar software de calidad.»

Ricardo Z 18 Agosto 2020

Proyecto Software

Desarrollar un software es como construir un edificio: hay mucho que hacer andes del "verdadero" trabajo.»

Ricardo Z 18 Agosto 2020

Proceso de Software

Es un conjunto estructurado de actividades, cuya meta es el desarrollo o evolución de un software eficaz y eficiente que reúna los requisitos del cliente.»

Ricardo Z 18 Agosto 2020

Problemas al desarrollar Software

Actualmente en los sistemas basados en computadoras, el software ha superado al hardware como factor decisivo de éxito. El software es el factor que marca la diferencia.»

Ricardo Z 18 Agosto 2020

Factores de Calidad del Software

Un software de calidad tiene concordacia con: Los requisitos funcionales y de rendimiento explícitamente estableccidos, los estándares de desarrollo explícitamente documentados y las características implícitas que se espera de todo software desarrollado profesionalmente.»

Ricardo Z 18 Agosto 2020

Modelo de Proceso de Software

Representación formal y simplificada de un proceso de software, presentada desde una perspectiva específica. Dicho de otra manera, los procesos son instancias de un modelo de proceso.»
